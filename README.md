# google-books 📚
has 3 main pages

| Pages  | Page Description  |
| ------------- |:-------------:|
| Home     | 🏡main page, you can search and filter here    |
| Book Information     | 📄to view the book information that your selected     |
| MyFavorite     | 💟 to view collections of books that you add to favorite    |
